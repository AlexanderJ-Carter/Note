# 我的学习笔记

使用 VitePress 构建的个人学习笔记站点，部署在 `note.alexander.xin`。

## 📚 内容

- **编程学习** - JavaScript、Python 等编程语言学习记录
- **技术笔记** - 开发经验和技术总结
- **工具使用** - Git、VS Code 等工具使用指南
- **项目经验** - 实际项目开发经验分享

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建静态文件
npm run docs:build
```

## 🌐 在线访问

[https://note.alexander.xin](https://note.alexander.xin)

## 🛠️ 技术栈

- VitePress - 静态站点生成器
- GitHub Pages - 免费托管
- GitHub Actions - 自动部署
